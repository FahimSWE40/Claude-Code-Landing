Step 01

Install Claude Code Globally

Claude Code is Anthropic's official agentic developer CLI. Install it globally on your machine using npm. Ensure you have Node.js installed.

bash
$ npm install -g @anthropic-ai/claude-code
# Fetching packages from npm registry...
+ @anthropic-ai/claude-code@latest installed successfully
✓

Verify the Installation

Verify that the Claude CLI is successfully installed on your path by running the version command:

$ claude --version
claude/0.2.x-beta (node/v20.11.0)

Step 03

Configure Environment Variables

To redirect Claude Code's traffic to AgentRouter, you need to set up three key environment variables. Select your operating system below to see the commands.

Run these setx commands in PowerShell to apply variables system-wide. After running, you must restart your terminal window for the changes to load.

PowerShell
PS> setx ANTHROPIC_BASE_URL "https://agentrouter.org/"
SUCCESS: Specified value was saved.
PS> setx ANTHROPIC_AUTH_TOKEN "sk-your-agentrouter-key"
SUCCESS: Specified value was saved.
PS> setx ANTHROPIC_API_KEY "sk-your-agentrouter-key"
SUCCESS: Specified value was saved.
⚠️ Important: You must completely close and open a brand new PowerShell window after running these commands to ensure the new environment variables are loaded.

Append these variables to your profile configuration file (typically ~/.bashrc, ~/.zshrc, or ~/.profile) and reload.

bash
# Add to ~/.bashrc or ~/.zshrc:
export ANTHROPIC_BASE_URL="https://agentrouter.org/"
export ANTHROPIC_AUTH_TOKEN="sk-your-agentrouter-key"
export ANTHROPIC_API_KEY="sk-your-agentrouter-key"
$ source ~/.bashrc
# (Or source ~/.zshrc depending on shell)

One-Line Setup (Append & Reload)

Copy and run this command to append variables to your configuration and reload automatically:

$ echo 'export ANTHROPIC_BASE_URL="https://agentrouter.org/"' >> ~/.bashrc && echo 'export ANTHROPIC_AUTH_TOKEN="sk-your-agentrouter-key"' >> ~/.bashrc && echo 'export ANTHROPIC_API_KEY="sk-your-agentrouter-key"' >> ~/.bashrc && source ~/.bashrc

Step 06

Troubleshooting Connection Errors

If things don't work right away, review these diagnostics to quickly identify and resolve auth issues.

🔌 Connection Diagnostics Helper Select Issue

Issue: 401 Unauthorized Error

This means your environment variable is either missing, loaded in a different terminal thread, or configured with an incorrect key.

1. Verify Current Environment Key

Run the query command to print the auth token loaded in the session:

Windows (PowerShell)
PS> echo $env:ANTHROPIC_AUTH_TOKEN
macOS / Linux
$ echo $ANTHROPIC_AUTH_TOKEN

It must display your API key. If it prints blank, your environment variable did not load correctly. Ensure you restarted your terminal.

Issue: Conflicts with Saved Anthropic Login

If you previously logged into Anthropic's cloud natively through Claude Code, the client might be trying to load cached cookies/tokens instead of reading environment variables.

Solution: Log Out and Restart

Remove existing Anthropic sessions and restart Claude to force environment variable loading:

$ claude logout
✓ Logged out successfully. Cached sessions removed.
$ claude
